== Members

[#tryLoadLayoutResource_Class_String]
=== tryLoadLayoutResource(Class, String)

Try to locate and load a xref:refguide:applib:index/layout/resource/LayoutResource.adoc[LayoutResource] by type and name.

Implementing beans may chose to be indifferent by returning an empty xref:refguide:commons:index/functional/Try.adoc[Try]

[#lookupLayoutResource_Class_String]
=== lookupLayoutResource(Class, String)

Optionally returns a xref:refguide:applib:index/layout/resource/LayoutResource.adoc[LayoutResource] based on whether it could be resolved by type and name and successfully read.

Silently ignores exceptions underneath, if any.
